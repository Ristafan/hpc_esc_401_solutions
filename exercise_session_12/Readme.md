### Step by step

1. Create instance on ScienceCloud (https://cloud.s3it.uzh.ch/project/instances/)
2. Connect to instance 
```bash

```

3. Install packages
```bash
sudo apt update
sudo apt install -q g++ make
```

4. Run file
```bash
./main
```
